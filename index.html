Bienvenidos a mi Página Web

Aprendiendo HTML de forma sencilla

Esta página es un ejemplo simple que utiliza las etiquetas fundamentales de HTML5, atributos comunes, imágenes, enlaces, listas, tablas y formularios.

Puedes explorar cómo se organiza el contenido y cómo se aplican los atributos para mejorar accesibilidad y funcionalidad.

Aprende más en W3Schools.

Elementos clave que usamos:

  • Títulos: <h1> a <h6>
  • Párrafos: <p>
  • Enlaces: <a>
  • Imagen: <img>
  • Listas: <ul>
  • Tablas: <table>
  • Formularios: <form>, <input>, <select>, <textarea>
